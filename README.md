
## 功能说明

- 左侧菜单导航
- 数字人名称输入
- 视频上传区域
  - 点击上传
  - 拖拽上传
  - 显示已上传文件名
- 注意事项卡片区
- 音色选择区
- 右侧案例预览区
- 响应式布局
  - 宽屏：左右布局
  - 窄屏：上下布局

## 技术栈

- Vue 3
- Vite

## 启动方式

先确保安装 Node.js。

```bash
npm install
npm run dev
```

默认启动后可在本地开发地址查看。

## 打包构建

```bash
npm run build
```
